AGENDAMENTO DE SALAS v0.3
Autores: Adriano José da Silva e Luiz Eugênio Espinosa Aranha

O software "Agendamento de salas" é parte integrante da atividade avaliativa do curso de Bacharelado em Ciência da Computação da Unioeste, campus de Foz do Iguaçu, que busca avaliar, entre outras coisas, o entendimento de conceitos aprendidos em aula na disciplina de Intelegência Artificial, através da análise e resolução do problema de alocação de salas de aula, utilizando estratégias de busca.

-------------------------------------------
Plataforma e Requisitos de Sistema

O software foi desenvolvido e testado para plataforma Windows/i386 e requer ambiente de execução java JRE instalado.

-------------------------------------------
Linguagem e Ambiente de Desenvolvimento

Desenvolvido na linguagem Java (JRE1.6.0_01) através do IDE "Eclipse SDK v3.3.2 build M20080221-1800". 

-------------------------------------------
Instalação e execução

Para executá-lo em ambiente gráfico, basta abrir o arquivo "TRABIA01-2010.jar". Para tanto deve-se estar com o ambiente de execução java (JRE) instalado.

-------------------------------------------
Documentação do código

Os métodos mais importantes no código fonte do programa foram comentados no padrão javadoc. Além disso, um diagrama de classe contendo as principais classes de negócio é disponibilizado na pasta "docs".
